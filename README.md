### 👨‍💻 `whoami`

> I translate complex problems into low-latency logic. As a third-year CS undergraduate, I operate at the intersection of raw algorithmic efficiency and AI orchestration. For me, a codebase isn't just a set of instructions—it is an ecosystem. Whether I am carving out micro-optimizations in C++ range queries, designing XML frameworks to force cognitive reasoning in LLMs, or mapping the deep architecture of Operating Systems, my baseline is always the same: **I build to understand the machine, and I optimize to master it.**

<br>

### ⚡ What I'm Working On

* 💻 **Competitive Programming:** Heavily focused on Data Structures & Algorithms (C++ / Codeforces). Currently refining purely functional implementations of Mo's Algorithm (bypassing class overhead for CP speed) and managing tail-query block deltas in Square Root Decomposition.
* 🤖 **AI & Prompting:** Exploring the capabilities of local and cloud LLMs (Ollama, DeepSeek, MiniMax) using Claude Code. I spend time experimenting with XML-based prompt frameworks to see how models handle logical parsing and complex reasoning.
* 🎓 **System Fundamentals:** Building comprehensive, highly-structured study artifacts for core engineering subjects like Operating Systems and Database Management Systems to maintain strong academic performance.
